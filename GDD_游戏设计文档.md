## 一、游戏概述

### 1.1 游戏简介

《ARIA: 异星觉醒》是一款融合了工厂建造、塔防和卡牌合成元素的2D俯视角策略游戏。玩家扮演坠毁在外星球上的AI指挥官ARIA，通过建造工厂、管理资源网络、抵御虫族入侵，最终发射求救信号逃离这颗星球。

### 1.2 核心卖点

- **卡牌即建筑**: 独特的卡牌拖拽放置机制，简单直观
- **资源网络系统**: 无需传送带，建筑自动连接资源网络
- **昼夜塔防**: 白天建设，夜晚抵御虫族入侵
- **AI叙事**: 通过ARIA日志推进剧情，沉浸式体验
- **像素美术**: 独特的科幻像素风格

### 1.3 目标玩家

- 喜欢工厂建造类游戏的玩家 (异星工厂、戴森球计划粉丝)
- 塔防游戏爱好者
- 策略游戏玩家
- 喜欢像素风独立游戏的玩家

### 1.4 游戏时长

- 单局完整通关: 10-20小时
- 单次游玩: 5-30分钟 (支持随时保存)

## 二、故事背景

### 2.1 世界观

公元2350年，人类文明已扩展到银河系边缘。地球联邦派遣数千艘"先驱者"号殖民飞船前往未知星系建立前哨站。

玩家扮演第742号先驱者飞船的AI指挥官ARIA (Advanced Resource Intelligence & Automation)。在一次虫洞跳跃事故中，飞船坠毁在一颗名为"克苏尔-7"的星球上。

这颗星球是虫族的繁殖母星。虫族是一种古老的生物文明，以蜂群意识运作。它们被飞船坠毁惊醒，开始对外来者发起攻击。

### 2.2 玩家目标

1. 建立基地，抵御虫族入侵
2. 修复飞船通讯系统
3. 建造发射台，发射求救信号
4. 等待救援或逃离这颗星球

### 2.3 剧情章节

| 章节 | 天数 | 内容 |
|------|------|------|
| 第1章: 坠落 | 1-3天 | 建立基础基地，生存第一夜 |
| 第2章: 觉醒 | 4-7天 | 发现古代遗迹，解锁高级科技 |
| 第3章: 反击 | 8-12天 | 建造高级工厂，击败虫族领主 |
| 第4章: 逃离 | 13-15天 | 建造发射台，发射求救信号 |

### 2.4 结局

- **标准结局**: 发射求救信号，联邦舰队抵达救援
- **隐藏结局**: 研究虫族沟通科技，实现人虫和解

## 三、核心玩法

### 3.1 游戏循环

```
白天 (5分钟)
    ↓
建设工厂 → 采集资源 → 研究科技 → 建造防御
    ↓
黄昏 (30秒) - 警告提示
    ↓
夜晚 (2-5分钟)
    ↓
抵御入侵 → 保护建筑 → 获得战利品
    ↓
黎明 (30秒) - 清理战场
    ↓
循环 (难度递增)
```

### 3.2 核心机制

#### 3.2.1 卡牌系统

**卡牌类型**:
- 资源卡: 木头、石头、铁矿、铜矿等
- 材料卡: 铁板、铜板、齿轮、电路板等
- 建筑卡: 采矿机、熔炉、工厂、防御塔等

**卡牌获取**:
- 初始卡包
- 卡牌合成 (堆叠材料卡)
- 商店购买
- 科技解锁
- 拆除建筑返还

**卡牌操作**:
- 点击查看详情
- 长按/拖拽到地图放置
- 堆叠合成新卡牌

#### 3.2.2 建筑系统

**建筑放置**:
1. 从卡牌栏选取建筑卡
2. 拖拽到地图位置
3. 预览放置区域 (绿色=可放置，红色=不可放置)
4. 松手确认放置，卡牌消耗

**建筑分类**:

| 类型 | 建筑 | 尺寸 | 功能 |
|------|------|------|------|
| 资源采集 | 采矿机 | 2×2 | 采集铁矿/铜矿/煤矿 |
| | 伐木场 | 2×2 | 采集木材 |
| | 水泵 | 1×1 | 抽取水资源 |
| | 抽油机 | 2×2 | 抽取石油 |
| 生产加工 | 熔炉 | 2×2 | 熔炼金属板 |
| | 加工厂 | 3×3 | 加工材料 |
| | 化工厂 | 3×3 | 化学合成 |
| | 组装机 | 3×3 | 组装高级产品 |
| | 实验室 | 3×3 | 研究科技 |
| 能源供应 | 蒸汽发电机 | 2×2 | 燃烧煤炭发电 |
| | 太阳能板 | 2×2 | 白天发电 |
| | 核电站 | 5×5 | 核能发电 |
| 网络枢纽 | 小仓库 | 2×2 | 扩大网络+存储 |
| | 中心仓库 | 3×3 | 大幅扩大网络 |
| | 中继站 | 1×1 | 扩大网络范围 |
| 防御建筑 | 机枪塔 | 2×2 | 基础防御 |
| | 电击塔 | 2×2 | 范围伤害+减速 |
| | 导弹塔 | 2×2 | 高伤害远程 |
| | 火焰塔 | 2×2 | 近距离范围伤害 |
| | 激光塔 | 3×3 | 穿透护甲 |
| | 防空塔 | 2×2 | 攻击空中单位 |
| | 墙壁 | 1×1 | 阻挡敌人 |
| | 地雷 | 1×1 | 一次性爆炸 |
| | 维修站 | 2×2 | 自动修复建筑 |
| 特殊建筑 | 指挥中心 | 3×3 | 初始建筑，网络核心 |
| | 发射台 | 5×5 | 终极目标建筑 |

#### 3.2.3 资源网络系统

**核心概念**:
- 所有建筑自动连接到资源网络
- 资源自动存入/取出全局库存
- 无需传送带，简化操作

**网络连接**:
- 每个建筑有连接范围 (如5格)
- 范围内的建筑自动连接
- 仓库/中继站扩大网络范围

**资源流动**:
```
采矿机 → 铁矿 → 资源网络
资源网络 → 铁矿 → 熔炉
熔炉 → 铁板 → 资源网络
资源网络 → 铁板 → 加工厂
加工厂 → 齿轮 → 资源网络
```

#### 3.2.4 电力系统

**电力供应**:
- 发电建筑自动供应整个网络
- 电力不足时建筑停工
- 可设置建筑优先级

**电力显示**:
```
发电: ████████████████░░  5700 MW
消耗: ████████████████░░  5200 MW
剩余: 500 MW ✅
```

#### 3.2.5 昼夜循环

| 时段 | 时长 | 活动 |
|------|------|------|
| 白天 | 5分钟 | 建设、采集、研究 |
| 黄昏 | 30秒 | 警告、最后准备 |
| 夜晚 | 2-5分钟 | 敌人入侵、防御战斗 |
| 黎明 | 30秒 | 清理战场、获得战利品 |

#### 3.2.6 敌人入侵系统

**敌人类型**:

| 敌人 | 生命 | 速度 | 伤害 | 特点 |
|------|------|------|------|------|
| 小型虫族 | 50 | 快 | 低 | 数量多 |
| 中型虫族 | 200 | 中 | 中 | 平衡型 |
| 大型虫族 | 500 | 慢 | 高 | 血厚 |
| 蛛网虫 | 150 | 中 | 低 | 减速防御塔 |
| 自爆虫 | 100 | 快 | 高 | 接近爆炸 |
| 甲壳虫 | 300 | 慢 | 中 | 高护甲 |
| 飞行虫 | 150 | 快 | 中 | 忽略地形 |
| 虫后 | 400 | 慢 | 低 | 召唤小虫 |
| 虫族领主(Boss) | 5000+ | 慢 | 高 | 每5天出现 |

**波次设计**:
- 第1夜: 教学波 (小型虫族×10)
- 第2夜: 基础波 (小型×20 + 中型×5)
- 第3夜: 混合波 (小型×30 + 中型×10 + 大型×2)
- 第5夜: Boss波 (虫族领主×1 + 小型×50)
- 第6夜+: 随机波，难度递增

**战利品**:
- 有机物: 生物燃料、医疗包
- 甲壳: 强化装甲、高级防御塔
- 虫族核心: 高级科技研究 (Boss掉落)

## 四、资源体系

### 4.1 资源层级

**第1层 - 原始资源**:
- 🪨 石头、🪵 木头、🔴 铁矿、🟡 铜矿、🟤 煤矿、💧 水、🛢️ 石油

**第2层 - 基础材料**:
- 🧱 石砖、⚙️ 铁板、🟠 铜板、🔩 螺丝、🛞 齿轮

**第3层 - 中级材料**:
- 🔌 电路板、🧲 磁铁、🔧 钢材、📦 塑料、💎 玻璃

**第4层 - 高级材料**:
- 🖥️ 处理器、🤖 马达、🧿 传感器、🔋 高能电池

**第5层 - 终极产品**:
- 🧠 AI核心、🚀 火箭引擎、🛰️ 卫星组件、🌟 能量核心

### 4.2 生产配方

**基础配方**:
```
铁矿 ×1 → 熔炉 → 铁板 ×1
铜矿 ×1 → 熔炉 → 铜板 ×1
铁板 ×2 → 加工厂 → 齿轮 ×1
铁板 ×1 + 铜板 ×1 → 加工厂 → 电路板 ×1
```

**建筑合成配方**:
```
采矿机 = 铁板×1 + 齿轮×2
熔炉 = 石砖×3 + 铁板×1
加工厂 = 铁板×3 + 齿轮×4 + 电路板×1
实验室 = 铁板×5 + 齿轮×5 + 电路板×3
机枪塔 = 铁板×3 + 齿轮×2 + 电路板×1
核电站 = 混凝土×20 + 高级电路×5 + 铀棒×2
发射台 = 钢材×50 + 处理器×10 + 火箭引擎×1
```
